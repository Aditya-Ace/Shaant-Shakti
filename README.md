## 📦 Installation

1. Go to Extensions (`Ctrl+Shift+X`) in VS Code
2. Search for `Shaant Shakti`
3. Click **Install**
4. Press `Ctrl+K Ctrl+T` and choose **Shaant Shakti** from the theme list

---
